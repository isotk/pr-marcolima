# Devocional Diario

App estatico de devocional diario em portugues.

## Publicacao

Produção: https://devocional-diario.vercel.app

## Recursos

- 3 devocionais visíveis (hoje, ontem, anteontem).
- Date badge vertical (mês, dia, dia da semana).
- Escritura com link direto para a Bíblia NVI.
- Termômetro Espiritual interativo (arrastável).
- Anotações por devocional salvas no navegador.
- Favoritos salvos no navegador.
- Busca por tema, referência ou palavra.
- Seções "O que é um Devocional Diário" e "Como fazer".

## Como abrir

Abra `index.html` no navegador.

Se o navegador bloquear `fetch` local, rode um servidor simples:

```bash
python3 -m http.server 8080
```

Depois acesse:

```text
http://localhost:8080/devocional-diario/
```
